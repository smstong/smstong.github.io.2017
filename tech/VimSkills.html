Basic commands

Basic settings

  • set tabstop=4
    • every time you press the TAB key, you will get a visual space width of 4 characters.
  • set expandtab
    • Every time you press the TAB key, you will get 4 spaces instead of a TAB.(Just the same as you press SPACE "tabstop" times.
  • set wrap
    • If a physical text line cannot be displayed in a screen line, the physical line will be wrapped into more than 1 screen lines.
