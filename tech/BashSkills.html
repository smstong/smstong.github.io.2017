daily commands

  • tar -cvpzf backup.tar.gz /var/www/html
    • -c : create
    -v : verbose
    -p : permission preserved
    -z : compressed with gzip
    -f : output to a file rather than stdout
  • tar -xvpzf backup.tar.gz -C /recover
    • -x : expand a tarball
    -v : verbose
    -p : permission preserved
    -z : uncompressed with gunzip
    -f : input from a file rather than stdin
    -C : change to desgined dicrectory rather than current directory
